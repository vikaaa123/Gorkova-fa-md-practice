Что такое Activity и зачем оно нужно?

Это базовый компонент приложения для андроида. Та часть приложения, с которой взаимодействует пользователь. 
Внутри Activity расположены дочерние элементы интерфейса.

Что такое View и зачем оно нужно?

Это элемент интерфейса (кнопка, поле для ввода текста и т.д)

Что такое Layout и зачем оно нужно?

Это наследники ViewGroup, контейнеры для View.
Используются для того чтобы вариативно распологать элементы View.

Почему все строки в Layout указываются косвенно из файла с ресурсами?

Чтобы максимально отделить локализованные аспекты приложения от функциональности основного приложения. 

Что такое версия API?

Это версия для Android, которая сообщает, на какую версию API ориентироваться для запуска приложения.
Обновления API фреймворка платформы разрабатываются таким образом, чтобы новая версия API оставалась совместимой с более ранними версиями, 
поэтому большинство изменений в новом API являются аддитивнымы.
